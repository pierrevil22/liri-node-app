# liri-node-app

LIRI will search Spotify for songs, Bands in Town for concerts, and OMDB for movies

Node packages crucial for this app

Node-Spotify-API

- Axios to grab data from the OMDB API and the Bands In Town API

- Moment

- DotEnv
